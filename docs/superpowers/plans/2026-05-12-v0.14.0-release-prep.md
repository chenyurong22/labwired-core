# LabWired Core v0.14.0 Release Prep Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Prepare, verify, tag, push, and publish LabWired Core `v0.14.0`.

**Architecture:** Keep the release prep as metadata and documentation changes. Use Cargo to refresh lockfile state, git to commit and tag, and GitHub CLI to publish release notes derived from `CHANGELOG.md`.

**Tech Stack:** Rust workspace, Cargo, MkDocs documentation files, git, GitHub CLI.
